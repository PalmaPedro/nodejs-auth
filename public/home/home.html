Project description

This is the second mandatory assignment - Basic login system

Extended features from the Auth project, from class:

  • Create a user account, using a form
  • Confirmation account email (Nodemailer). Only after confirmation of email will the account become active
  • Database to store user details
  • User can login and have access to his personal webpage - it will only display his personal data, for demonstration purposes
  • This personal is only accessible if user is logged in

Technologies used in this project:

  • express
  • objection
  • knex
  • express-session
  • express-rate-limit
  • bcrypt
  • nodemailer
  • mysql
